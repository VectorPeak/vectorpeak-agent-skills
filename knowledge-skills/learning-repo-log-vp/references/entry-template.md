# Learning Log Entry Template

Use this template for normal entries:

```markdown
## YYYY-MM-DD：概念名称

### 一句话理解

用自己的话解释这个概念是什么。

### 在 {{project_name}} 中怎么用

说明它在当前项目中对应哪个模块、流程、文件、配置或设计决策。

### 为什么值得记录

说明它解决了什么问题，为什么容易被问到，或者为什么后续开发会再次遇到。

### 易混淆点

列出相近概念、常见误解、边界条件或使用陷阱。

### 后续复习问题

- 问题一
- 问题二
```

Use this compact template for small notes:

```markdown
## YYYY-MM-DD：概念名称

- **一句话理解：** ...
- **项目关联：** ...
- **后续复习：** ...
```

Use this addendum format when updating an existing entry:

```markdown
### YYYY-MM-DD 补充

- 新增理解：
- 项目变化：
- 待复习问题：
```
