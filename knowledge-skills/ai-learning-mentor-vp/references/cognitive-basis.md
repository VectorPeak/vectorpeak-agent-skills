# Cognitive Basis

Use this reference only when the user asks why the learning method works, wants scientific grounding, or asks for a more formal learning design.

## Memory Aid

Core mechanism: convert a slippery concept into chunks, cues, contrasts, and retrieval prompts.

Cognitive basis:

- Chunking reduces working-memory load.
- Retrieval cues make later recall easier.
- Contrastive encoding separates similar concepts and reduces confusion.
- Spaced recall strengthens long-term retention.

Operational implication: produce minimal chunks, hooks, contrast tables, and 24h/3d/7d review prompts.

## AI As Student / Feynman Core

Core mechanism: the learner explains a concept to AI; AI acts as a confused but precise listener and interviewer.

Cognitive basis:

- Generation effect: explaining in one's own words improves encoding.
- Self-explanation effect: being forced to clarify vague causal links exposes hidden gaps.
- Teach-to-learn expectation: preparing to teach improves organization and recall.

Operational implication: ask the learner to output first, then diagnose.

## Analogy Generator

Core mechanism: map an abstract concept to familiar concepts, then mark where the analogy fails.

Cognitive basis:

- Dual coding: verbal + imagistic/structural representations improve memory.
- Analogical reasoning: comparing structures helps transfer beyond surface features.
- Structural mapping: good analogies preserve relations, not just labels.

Operational implication: always include failure boundaries.

## Relationship Graph Builder

Core mechanism: connect concepts across modules and domains.

Cognitive basis:

- Schema construction: concepts become usable when relations are explicit.
- Far transfer: structural similarity across different domains supports flexible problem solving.

Operational implication: include prerequisites, siblings, opposites, downstream uses, and similar structures.

## Deliberate Practice Coach

Core mechanism: generate progressively harder variants targeting a known weak point.

Cognitive basis:

- Deliberate practice: improvement comes from targeted work on weaknesses, not repetition alone.
- Immediate feedback: fast correction prevents error patterns from fossilizing.
- Spaced repetition: review timing improves long-term retention.

Operational implication: ask first, grade after, adapt next.

## Knowledge Compression

Core mechanism: compress a topic into a framework and active-recall cards.

Cognitive basis:

- Active recall: retrieving strengthens memory more than rereading.
- Testing effect: recall attempts improve long-term retention.
- Cognitive load reduction: short cards reduce review friction.

Operational implication: generate short cards with anti-confusion points.
