# Project Concept Explainer

项目伴读与技术概念讲解 skill。它用于在做项目时解释关键技术名词、模型算法、配置项、数据字段、架构决策，并在需要时生成项目知识地图、术语表、学习笔记和研究笔记

## 适用场景

- 解释当前项目里的关键技术名词
- 比较两个容易混淆的概念
- 扫描项目并生成知识地图
- 生成 glossary / learning notes / research notes
- 把 README、docs、scripts、configs、notebooks、reports 里的概念讲清楚

## 示例请求

```text
使用 project-concept-explainer，解释这个项目里的 BM25、FAISS、Hard Negative Mining
```

```text
使用 project-concept-explainer，扫描当前项目并生成一份项目知识地图
```

```text
使用 project-concept-explainer，把这个仓库的核心术语整理成 docs/glossary.md
```
